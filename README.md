# The Serenade Town

Single-page site for The Serenade Town (Andheri / Ghatkopar). Built with Next.js and Tailwind.

**After cloning:** run `npm install`, then `npm run dev`. Do not commit `node_modules` (it's in `.gitignore`).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deploy on Vercel

Deploy with [Vercel](https://vercel.com/new) or see [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying).
