# The Serenade Town ✨

**Your moment, perfectly celebrated.**

A single-page website for [The Serenade Town](https://github.com/sakshamtikekar19/the-serenade-town) — private theater celebrations in **Andheri East** and **Ghatkopar**, Mumbai. Built with Next.js and Tailwind CSS.

---

## Quick start

After cloning the repo:

```bash
npm install
npm run dev
```

Then open **http://localhost:3000** in your browser.

> Do not commit `node_modules` — it's already in `.gitignore`.

---

## What's in the site

- **Branch toggle** — Andheri East / Ghatkopar (directions & WhatsApp by branch)
- **Occasions** — Birthday, Anniversary, Proposal, Movie date, and more
- **Packages** — Silver, Golden, Diamond with weekday/weekend pricing
- **Price calculator** — Live total with add-ons (cake, fog, extra guests)
- **Book via WhatsApp** — Pre-filled message with package and total
- **Mobile responsive** — No horizontal scroll, works on all screen sizes

---

Deploy on [Vercel](https://vercel.com/new) (recommended for Next.js) or follow [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying).
